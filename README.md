## 專案說明
* RWD與SCSS切版
* 使用 axios 串接 API
* 使用 Git 版本控制

### 使用技術/相關套件
* SCSS
* Swiper
* axios
* D3、C3
* SweetAlert
* validate
* anime

1. API 的應用，處理CRUD資料
1. 資料數據圖表化
1. validation 表單驗證
1. loading 開場畫面
1. 陣列、物件資料處理
